## Purpose

Volume 3 translates Volume 2 business architecture into **domain-level functional behavior**—how each document type behaves, validates, transitions, and hands off to the next stage.

Volume 4 (Workflow Engine) implements State Machines from this volume. Volume 6 (UX Standards) projects Dashboard, Workspace, and Control Tower from domain rules here.
